OnShape Log In Documentation

Date : October 20, 2022

Product Name : OnShape CAD Browser

Author : Taylor Luamba

Version : 2014 - Present

Product Overview :

On-shape is an industry grade cloud-based computer aided design platform that employers and learners can access on any device, anywhere, and anytime.

Objectives:

Learners should be able to create their own account.

  1. Create a Student Account.

Phase 1: Access the website

Using internet browser, click into the address bar and input “onshape.com/en/education”.

In the top right corner, select the “Create a Student Account” button.

Phase 2: Inputting User Data

In the input box on the right, input user data.

Type in First name.

Type in Last name.

Select user employment status from the drop down menu.

Select user school level.

Click in the empty check box to acknowledge personal details will be process with appropriate privacy policy of OnShape.

Click on the dark green box with, “create account” written atop.

Phase 3: Sign up for OnShape’s Education Plan

In the input box, type user email address data.

Type in user affiliation’s name.

Input a hyperlink to user affiliation’s formal website.

Open a separate tab by clicking the plus sign on browser.

Search affiliation in the address bar.

Find appropriate website link for affiliation.

Right click on the link.

Select “copy link address” from dialog box.

Return to phase 3 tab.

Right click in empty space for affiliation url.

Select “paste” from dialog box.

Type in the city of user affiliation.

Type in the state shorthand for user affiliation.

Select affiliation’s country of residence.

Type in user title. (skip this is selected the student status in phase 2.)

Type in user mobile phone number (skip this is selected the student status in phase 2.)

Type in response to “What do you teach?”

Select “yes” or “no” for the bold font posed question.

Type in response to “What are you using OnShape for?”

Select the check boxes as they apply to you.

Select the blue button with “create account” written in white font.

Phase 4: Activate OnShape Account

Access your email mailbox used in phase 3.

Open a new tab.

Open Outlook 365, Gmail, Yahoo, Aol, etc.

Log in with valid credentials.

In user inbox, navigate to email from OnShape, with the subject line “Your OnShape Account is ready”.

Select the email.

Select the blue button with “ activate your account” written in white font.

A new tab will open which will allow user to create a password.

Create a password that meets the following requirements.

10 characters

1 number

1 lowercase

1 uppercase

Confirm your password.

Select the checkbox to validate understanding of Onshape Terms and Privacy Policy

Phase 5: Logging into New Account

Using internet browser, click into the address bar and input “onshape.com/en/education”.

In the top right corner, select “sign in”.

Input user email and password data previously selected.

Select the green button with “sign in” written atop.

During first log in, user is instructed to select units of measurement.

After selection, user views list of past work.
